# AI Voice-Enabled Calculator

An AI-based Voice-Enabled Calculator built using Python, Speech Recognition, and pyttsx3 libraries.

## 📌 Features

- Voice input for calculations
- Speech output for results
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Interactive voice assistant

## 🛠 Technologies Used

- Python
- speech_recognition
- pyttsx3
- pyaudio
